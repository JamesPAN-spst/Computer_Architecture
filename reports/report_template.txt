  §3  自评检查清单（Self-Evaluation Checklist）
═══════════════════════════════════════════════════════════════════════

  请逐项打 ✅ 或 ❌，并在不合格项说明原因。

  ── 3.1 教学风格 ──────────────────────────────────────────────────

  [ ] 3.1.1 每个 KP 包含 5 张 step-card：
        motivation / abstraction / derivation / application / example
  [ ] 3.1.2 三段式引导：为什么需要 → 怎么工作 → 系统中角色
  [ ] 3.1.3 没有无动机的知识清单（禁止填鸭式罗列）
  [ ] 3.1.4 没有跳过推导步骤（公式都要"长"出来）
  [ ] 3.1.5 没有用模糊类比代替精确描述
  [ ] 3.1.6 每个 KP 标注了 ⚠️ 易错点和难点（warning-box）
  [ ] 3.1.7 器件类内容使用了 Canvas/SVG 交互动画
  [ ] 3.1.8 章末有 8 道综合练习题（QUESTIONS 数组）
  [ ] 3.1.9 每个 KP 内有一道 inline-quiz（Quick Check）

  ── 3.2 技术规范 ──────────────────────────────────────────────────

  [ ] 3.2.1 纯 HTML + CSS + JS，无框架依赖
  [ ] 3.2.2 正确引用 shared/ 下的 CSS 和 JS 文件
  [ ] 3.2.3 CSS 路径使用正确的相对路径（../shared/ 或 ../../shared/）
  [ ] 3.2.4 MathJax 3 配置正确（tex-mml-chtml）
  [ ] 3.2.5 页面可在 GitHub Pages 静态部署
  [ ] 3.2.6 响应式布局正确（断点 900/600/380px）

  ── 3.3 HTML 结构 ──────────────────────────────────────────────────

  [ ] 3.3.1 章节页使用 class="chapter-page"
  [ ] 3.3.2 概览页使用 class="overview-page"
  [ ] 3.3.3 topbar 结构正确：brand → module info → right(chapter-indicator + theme-toggle)
  [ ] 3.3.4 toc-sidebar 包含所有 KP 锚点 + 综合练习锚点
  [ ] 3.3.5 chapter-hero 包含 chapter-label 和 h1
  [ ] 3.3.6 每个 kp-section 有正确的 id（kp1, kp2, ...）
  [ ] 3.3.7 step-card 使用正确的 class（step-motivation, step-abstraction,
        step-derivation, step-application, step-example）
  [ ] 3.3.8 kp-separator (<hr>) 在每个 KP 之间
  [ ] 3.3.9 exercises section 使用 id="exercises"
  [ ] 3.3.10 章节导航（chapter-nav）prev/next 链接正确

  ── 3.4 SVG/Canvas 规范 ────────────────────────────────────────────

  [ ] 3.4.1 SVG 包裹在 <figure class="kp-figure">
  [ ] 3.4.2 SVG viewBox="0 0 400 240" 或合理尺寸
  [ ] 3.4.3 CSS 类使用 .fg / .fg2 / .line / .accent
  [ ] 3.4.4 Canvas 动画有 Play/Pause 按钮
  [ ] 3.4.5 支持暗色/亮色主题适配

  ── 3.5 内容质量 ──────────────────────────────────────────────────

  [ ] 3.5.1 语言：中文为主，关键术语标注英文
  [ ] 3.5.2 公式渲染：所有数学公式使用 MathJax 语法
  [ ] 3.5.3 内容准确性：概念、公式、推导无科学错误
  [ ] 3.5.4 练习题有详细解答和易错点分析
  [ ] 3.5.5 inline-quiz 答案和反馈正确

  ── 3.6 data 属性 ──────────────────────────────────────────────────

  [ ] 3.6.1 body data-category 对应正确的 stage 编号
  [ ] 3.6.2 body data-module 对应正确的 chapter 编号
  [ ] 3.6.3 --domain-color 使用正确的 domain CSS 变量
