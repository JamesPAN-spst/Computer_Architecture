电容与电感

01电容（Capacitance）——存储电场能量

🎯 为什么研究这个问题

电阻只能消耗能量（转化为热），但很多时候我们需要存储能量并在需要时释放。电容器（Capacitor）就是这样一种器件——它通过在两块金属板之间建立电场来存储能量。

在计算机中，电容无处不在：DRAM 用微小的电容来存储每一个 bit 的数据（有电荷 = 1，无电荷 = 0）；CPU 供电电路中的去耦电容（Decoupling Capacitor）像"储水池"一样平滑电压波动；触摸屏利用人体改变电容值来检测触摸位置。

理解电容的原理，是理解存储、滤波和时序电路的基础。

🔍 抽象化方法

电容器的基本结构是两块相互平行的导体板（极板），中间隔着绝缘材料（介质，Dielectric）。当在两个极板之间施加电压时，正电荷聚集在一块板上，等量的负电荷聚集在另一块板上，两板之间形成电场。

我们定义电容为存储电荷的能力：

\[C = \frac{Q}{V}\]

其中 \(Q\) 是极板上的电荷量，\(V\) 是两板间的电压。单位是法拉（Farad, F），1 F = 1 C/V。实际电容器通常在 pF（\(10^{-12}\)）到 mF（\(10^{-3}\)）的量级。

📐 公式推导 / 模型建立

平行板电容器的推导：

两块面积为 \(A\) 的平行极板，间距为 \(d\)，中间充满介电常数为 \(\varepsilon\) 的介质。

由高斯定律，均匀带电平板产生的电场为 \(E = \sigma/\varepsilon = Q/(\varepsilon A)\)。

两板之间的电压：\(V = Ed = Qd/(\varepsilon A)\)。

因此：

\[C = \frac{Q}{V} = \frac{\varepsilon A}{d}\]

关键洞察：电容与板面积 \(A\) 成正比，与间距 \(d\) 成反比。要增大电容，就要增大面积或减小间距——这解释了为什么芯片中的电容结构不断向更薄的介质层和更高介电常数的材料发展。

储存的能量：

充电过程中，电压从 0 增长到 \(V\)。在电压为 \(v\) 时移动 \(dq\) 电荷需要做功 \(dW = v \cdot dq = v \cdot C \, dv\)。

\[W = \int_0^V Cv \, dv = \frac{1}{2}CV^2\]
⚠️ 电容的能量是 \(\frac{1}{2}CV^2\)，不是 \(CV^2\)！充电过程中电压在不断增加，平均电压只有 \(V/2\)，所以总功是 \(QV/2 = \frac{1}{2}CV^2\)。这个 \(\frac{1}{2}\) 系数也出现在弹簧势能 \(\frac{1}{2}kx^2\) 中——两者的数学结构完全相同。
🌐 实际应用

DRAM 存储：每个 DRAM 单元由一个电容和一个晶体管组成。电容充电代表"1"，放电代表"0"。但电容会通过泄漏电流逐渐放电，所以 DRAM 需要定期刷新（Refresh），这也是"动态"（Dynamic）的含义。

去耦电容：CPU 在高速切换时会产生瞬间大电流需求。去耦电容放置在 CPU 旁边，像"小水库"一样在需要时快速释放电荷，平滑供电电压。

触摸屏：电容触摸屏由多层透明导体组成。手指接近时改变了局部电容值，控制器通过检测电容变化来确定触摸位置。

✏️ 例题分析与易错点

例题：一个平行板电容器，板面积 \(A = 100\,\text{cm}^2\)，间距 \(d = 1\,\text{mm}\)，介质为空气（\(\varepsilon_0 = 8.85 \times 10^{-12}\,\text{F/m}\)）。求电容值。若充电到 100V，储存了多少能量？

❌ 常见错误：面积单位忘记转换——\(100\,\text{cm}^2\) 不是 \(100 \times 10^{-2}\,\text{m}^2\)，而是 \(100 \times 10^{-4}\,\text{m}^2 = 10^{-2}\,\text{m}^2\)。

✅ 正确解法：

\(C = \varepsilon_0 A/d = 8.85 \times 10^{-12} \times 10^{-2} / 10^{-3} = 88.5\,\text{pF}\)

\(W = \frac{1}{2}CV^2 = \frac{1}{2} \times 88.5 \times 10^{-12} \times 10^4 = 0.443\,\mu\text{J}\)

将平行板电容器的极板间距减半，电容值会？

02RC 电路（RC Circuit）——时间的尺度

🎯 为什么研究这个问题

前面我们讨论了电阻和电容各自的性质。但当它们连接在一起时，会产生一种全新的行为——时间依赖的响应。电阻和电容的组合（RC 电路）引入了"时间"这个维度。

在数字电路中，信号从 0 变到 1 不是瞬间完成的——它需要对线路上的寄生电容（Parasitic Capacitance）充电。这个充电过程的速度由 RC 时间常数决定。RC 延迟是限制芯片速度的核心因素之一。

每当你看到"充电曲线"、"上升时间"、"延迟时间"这些概念时，背后都是 RC 电路在起作用。

🔍 抽象化方法

考虑一个简单的 RC 充电电路：电压源 \(V_0\) 通过电阻 \(R\) 给电容 \(C\) 充电。

关键洞察：电容电压不能突变（因为 \(V = Q/C\)，电荷不能瞬间移动），所以充电是一个渐进的过程。

我们定义时间常数：

\[\tau = RC\]

它的物理意义是：电容电压从 0 上升到最终值的 63.2%（即 \(1 - 1/e\)）所需的时间。\(\tau\) 越大，充电越慢。

📐 公式推导 / 模型建立

从 KVL（基尔霍夫电压定律）出发：

回路中：电源电压 = 电阻上的电压 + 电容上的电压

\[V_0 = V_R + V_C = IR + V_C\]

又因为 \(I = C \frac{dV_C}{dt}\)，代入得：

\[V_0 = RC\frac{dV_C}{dt} + V_C\]

这是一阶线性常微分方程。初始条件 \(V_C(0) = 0\)，解为：

\[V_C(t) = V_0\left(1 - e^{-t/RC}\right) = V_0\left(1 - e^{-t/\tau}\right)\]

充电电流随时间指数衰减：

\[I(t) = \frac{V_0}{R}e^{-t/\tau}\]

关键时间节点：

  • \(t = \tau\)：\(V_C \approx 63.2\%\) 的 \(V_0\)
  • \(t = 3\tau\)：\(V_C \approx 95.0\%\) 的 \(V_0\)
  • \(t = 5\tau\)：\(V_C \approx 99.3\%\) 的 \(V_0\)（工程上视为"充满"）
⚠️ RC 时间常数的单位确实是秒！容易怀疑 \(\Omega \times F\) 的量纲，验证：\(\Omega \cdot F = (V/A)(C/V) = C/A = C/(C/s) = s\)。另外，\(\tau = RC\) 只适用于一阶 RC 电路。有多个电容和电阻时需要更复杂的分析。
🌐 实际应用

芯片中的 RC 延迟：现代芯片的互连线（Interconnect）有电阻和寄生电容。信号传播延迟近似为 \(t_d \approx 0.7RC\)（10%-90% 上升时间）。随着制程缩小，线变细（R 增大）且更密（C 增大），RC 延迟成为比逻辑门延迟更严重的瓶颈。

定时器：经典的 555 定时器芯片就是利用 RC 充放电来产生精确的时间延迟和方波信号。

滤波器：RC 低通滤波器让低频信号通过、衰减高频信号。截止频率 \(f_c = 1/(2\pi RC)\)。这在音频处理和信号调理中广泛使用。

✏️ 例题分析与易错点

例题：一根芯片内的铜走线，等效电阻 \(R = 500\,\Omega\)，寄生电容 \(C = 0.2\,\text{pF}\)。求信号从 10% 上升到 90% 所需的时间。

❌ 常见错误：直接用 \(\tau = RC\) 作为上升时间。\(\tau\) 是从 0% 到 63.2%，不是 10%-90%。

✅ 正确解法：

\(\tau = RC = 500 \times 0.2 \times 10^{-12} = 100\,\text{ps}\)

10%-90% 上升时间 \(t_r = \ln(9) \times \tau \approx 2.2 \times 100\,\text{ps} = 220\,\text{ps}\)

推导：\(0.1 = 1-e^{-t_1/\tau}\) 得 \(t_1 = 0.105\tau\)；\(0.9 = 1-e^{-t_2/\tau}\) 得 \(t_2 = 2.303\tau\)。\(t_r = t_2 - t_1 \approx 2.2\tau\)。

⚡ 快速检测
将 RC 电路中的电阻 R 加倍、电容 C 减半，时间常数 τ 如何变化？

03电感（Inductance）——存储磁场能量

🎯 为什么研究这个问题

电容通过电场存储能量。自然界中是否有对称的元件——通过磁场存储能量？答案就是电感器（Inductor）。

当电流通过线圈时，产生的磁场会"记住"这个电流。如果你试图快速改变电流，电感会产生感应电动势来"反抗"这个改变——这就是楞次定律（Lenz's Law）的体现。

在计算机电源中，电感是开关电源（Switching Power Supply）的核心组件，负责在 12V 输入和 1V CPU 核心电压之间高效转换。没有电感，就没有高效的供电系统。

🔍 抽象化方法

电感器通常是一个线圈。通过线圈的电流 \(I\) 产生磁通量 \(\Phi\)。电感定义为：

\[L = \frac{N\Phi}{I}\]

其中 \(N\) 是线圈匝数，\(\Phi\) 是每匝的磁通量。单位是亨利（Henry, H）。

电感的核心特性：它抗拒电流的变化。电感两端的电压与电流的变化率成正比：

\[V_L = L\frac{dI}{dt}\]

这与电容的 \(I = C \, dV/dt\) 形成完美的对偶关系。

📐 公式推导 / 模型建立

从法拉第定律出发：

法拉第电磁感应定律告诉我们，变化的磁通量会感应出电动势（EMF）：

\[\mathcal{E} = -N\frac{d\Phi}{dt}\]

又因为 \(L = N\Phi/I\)，即 \(N\Phi = LI\)，所以：

\[\mathcal{E} = -\frac{d(LI)}{dt} = -L\frac{dI}{dt}\]

负号表示感应电压反抗电流变化（楞次定律）。在电路分析中我们常写成 \(V_L = L\frac{dI}{dt}\)。

存储的能量：

给电感通电的过程中，功率 \(P = V_L \cdot I = LI\frac{dI}{dt}\)。积分得：

\[W = \int_0^I LI' \, dI' = \frac{1}{2}LI^2\]

与电容能量 \(\frac{1}{2}CV^2\) 完美对称。

⚠️ 电感电流不能突变！就像电容电压不能突变一样。如果强行在极短时间内切断电感电流（\(dI/dt \to \infty\)），\(V_L = L \cdot dI/dt\) 会产生巨大的电压尖峰。这就是为什么继电器断开时会产生电弧——存储在磁场中的能量必须释放。实际电路中通常并联二极管（续流二极管，Flyback Diode）来提供电流通路。
🌐 实际应用

开关电源（Buck Converter）：你的笔记本电源适配器将 220V 交流转为低压直流。核心拓扑中，开关（MOSFET）以高频（100kHz–MHz）通断，电感平滑电流，电容平滑电压，实现高效降压。效率可达 90%+ 以上。

EMI 滤波器：电感在高频时阻抗高（\(Z_L = j\omega L\)），可以阻挡高频噪声。电脑主板上常见的磁珠（Ferrite Bead）就是小型电感，用于滤除高频干扰。

无线充电：无线充电利用两个靠近的线圈之间的互感（Mutual Inductance）来传输能量，本质上是变压器原理。

✏️ 例题分析与易错点

例题：一个 \(10\,\text{mH}\) 的电感中通过 \(1\,\text{A}\) 电流。若在 \(10\,\mu\text{s}\) 内将电流减小到零，感应电压是多少？

❌ 常见错误：忘记毫亨（mH）和微秒（μs）的单位换算，导致数量级错误。

✅ 正确解法：

\(V_L = L \frac{dI}{dt} = 10 \times 10^{-3} \times \frac{1}{10 \times 10^{-6}} = 1000\,\text{V}\)

一个小小的 10mH 电感，电流变化率足够大时，就能产生 1000V 的高压！这解释了为什么电感断路保护如此重要。

04LC 振荡（LC Oscillation）——能量的摇摆

🎯 为什么研究这个问题

电容存储电场能量，电感存储磁场能量。如果我们把它们连在一起会怎样？答案令人着迷：能量会在电场和磁场之间反复转换，产生振荡。

这就像一个弹簧挂着的重物——势能和动能不断互换，产生往复运动。LC 振荡是最纯粹的电磁振荡形式，是所有无线通信、时钟信号生成的理论基础。

你手机里的射频芯片、CPU 的时钟振荡器、WiFi 路由器——它们的心脏都是某种形式的 LC 振荡电路。

🔍 抽象化方法

考虑一个理想 LC 电路（无电阻）：电容 \(C\) 初始充电到电压 \(V_0\)，然后接上电感 \(L\)。

初始状态：所有能量在电容中（\(W = \frac{1}{2}CV_0^2\)），电感电流为零。

然后电容开始放电，电流增大，电感磁场增强。当电容完全放电时，所有能量转移到电感中（\(W = \frac{1}{2}LI_{\max}^2\)）。

接着电感的磁场开始给电容反向充电……如此循环往复。

📐 公式推导 / 模型建立

从 KVL 出发：

LC 回路中：\(V_C + V_L = 0\)，即：

\[\frac{Q}{C} + L\frac{dI}{dt} = 0\]

由于 \(I = dQ/dt\)（约定放电为正）：

\[\frac{Q}{C} + L\frac{d^2Q}{dt^2} = 0\]

整理为标准形式：

\[\frac{d^2Q}{dt^2} + \frac{1}{LC}Q = 0\]

这是简谐振动方程！与弹簧振子 \(\ddot{x} + \omega^2 x = 0\) 完全同构。

解为：

\[Q(t) = Q_0 \cos(\omega_0 t + \varphi)\]

其中角频率：

\[\omega_0 = \frac{1}{\sqrt{LC}}\]

谐振频率：

\[f_0 = \frac{1}{2\pi\sqrt{LC}}\]

力学-电学对偶：质量 \(m \leftrightarrow L\)，弹性系数 \(k \leftrightarrow 1/C\)，位移 \(x \leftrightarrow Q\)，速度 \(v \leftrightarrow I\)。

⚠️ 理想 LC 振荡永不停止，但实际电路中总有电阻损耗。任何真实电路都有导线电阻和辐射损耗，使振荡幅度逐渐衰减（RLC 阻尼振荡）。要维持振荡，需要通过正反馈（如晶体管放大器）持续补充能量。这就是振荡器（Oscillator）的工作原理。
🌐 实际应用

时钟信号：CPU 时钟由石英晶体振荡器（Crystal Oscillator）产生。石英晶体等效为一个极高品质因数（Q factor）的 LC 电路，谐振频率极其稳定（ppm 级精度）。典型的基频是 25–50 MHz，再通过锁相环（PLL）倍频到 GHz 级别。

无线通信：发射机的 LC 振荡产生载波（Carrier），接收机的 LC 谐振回路选择特定频率的信号。调谐——改变 C 或 L 来改变 \(f_0\)——就是"调台"的原理。

RFID/NFC：近场通信芯片中的线圈和电容构成谐振电路，在特定频率（如 13.56 MHz）上高效耦合能量和数据。

✏️ 例题分析与易错点

例题：设计一个谐振在 \(f_0 = 1\,\text{MHz}\) 的 LC 电路。已知电感 \(L = 100\,\mu\text{H}\)，求所需电容。

❌ 常见错误：忘记 \(f = 1/(2\pi\sqrt{LC})\) 中 \(LC\) 在根号下，解方程时漏掉平方步骤。

✅ 正确解法：

由 \(f_0 = \frac{1}{2\pi\sqrt{LC}}\) 得 \(LC = \frac{1}{(2\pi f_0)^2}\)。

\(C = \frac{1}{(2\pi \times 10^6)^2 \times 100 \times 10^{-6}} = \frac{1}{3.948 \times 10^{13} \times 10^{-4}} = \frac{1}{3.948 \times 10^{9}} \approx 253\,\text{pF}\)

验证：\(f = 1/(2\pi\sqrt{100 \times 10^{-6} \times 253 \times 10^{-12}}) \approx 1\,\text{MHz}\) ✓
